# Wireless Communication Principles

A topic-wise organized reference and implementation repo for core wireless communication concepts.

---

## Structure

| Folder | Topics Covered |
|--------|---------------|
| [01_Signal_Fundamentals](01_Signal_Fundamentals/) | Signals, spectrum, Fourier analysis, sampling theorem, noise |
| [02_Modulation_Techniques](02_Modulation_Techniques/) | AM, FM, BPSK, QPSK, QAM, constellation diagrams |
| [03_Channel_Models](03_Channel_Models/) | AWGN, Rayleigh/Rician fading, path loss, shadowing |
| [04_OFDM](04_OFDM/) | Cyclic prefix, subcarriers, IFFT/FFT, PAPR |
| [05_MIMO](05_MIMO/) | Spatial multiplexing, beamforming, diversity, capacity |
| [06_Error_Coding](06_Error_Coding/) | Hamming, convolutional, turbo, LDPC, polar codes |
| [07_Multiple_Access](07_Multiple_Access/) | TDMA, FDMA, CDMA, OFDMA, NOMA |
| [08_5G_NR](08_5G_NR/) | NR numerology, mmWave, massive MIMO, network slicing |
| [09_Link_Budget](09_Link_Budget/) | SNR, BER, receiver sensitivity, margin calculations |
| [10_Simulations](10_Simulations/) | End-to-end simulation scripts (Python/MATLAB) |

---

## Goals
- Implement and visualize each concept from scratch
- Connect theory to simulation
- Build intuition for system-level design
